## Operators
```js
// Unary
typeof

// Binary
// Mathmatical Operators
+ // Add
- // Subtract
/ // Divide
* // Multiply
% // Remainder

// Comparison Operators
=== // is equal
!== // is not equal
>   // is bigger than
<   // is smaller than
>=  // is bigger or equals
<=  // is smaller or equals

// Logical
&& // and
|| // or
```

## Helpful way to think about Logical Operators

```js
(true && false && true)
// 1 x 0 x 1 = 0, therefor false

(true && true && true)
// 1 x 1 x 1 = 1, therefor true

(true || false || true)
// 1 + 0 + 1 = more than 0, therefore true

(false || false || false)
// 0 + 0 + 0 = 0, therefore false

```
